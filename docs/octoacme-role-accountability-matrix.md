# OctoAcme — Role Accountability Matrix

Use this lightweight matrix to confirm who is accountable, who contributes, and where handoffs happen across common project activities.

## How to use
- Keep one clearly named accountable owner for each activity.
- Confirm collaborators before kickoff and revisit during planning changes.
- Use the "Handoff / Key Sync" column to reduce gaps between teams.

## Matrix

| Activity | Accountable Role | Key Collaborators | Handoff / Key Sync |
| --- | --- | --- | --- |
| Define problem statement and success metrics | Product Manager | Business Analyst, Stakeholders, Project Manager | PdM shares approved scope and outcomes with PM + delivery team |
| Plan timeline, milestones, and dependencies | Project Manager | Product Manager, Scrum Master, Developers | PM publishes plan and risk owners before sprint execution |
| Refine requirements and acceptance criteria | Business Analyst | Product Manager, Developers, QA/Testing, UX Designer | BA + PdM finalize acceptance criteria before implementation starts |
| Design user flows and usability expectations | UX Designer | Product Manager, Developers, QA/Testing | UX walkthrough with engineering + QA before build completion |
| Build and review implementation | Developers | UX Designer, Business Analyst, QA/Testing, DevOps Engineer | PR includes requirement/design links and test notes for QA handoff |
| Validate quality and acceptance | QA/Testing | Developers, Product Manager, UX Designer | QA confirms pass/fail with PdM + Dev before release decision |
| Prepare and execute release | DevOps Engineer | Developers, QA/Testing, Project Manager, Product Manager | Go/no-go checkpoint and rollout/rollback confirmation |
| Communicate status, risks, and escalations | Project Manager | Product Manager, Scrum Master, Stakeholders | Weekly status update with explicit owners and due dates |
| Facilitate ceremonies and improvement actions | Scrum Master / Agile Facilitator | Project Manager, Product Manager, Developers, QA/Testing | Retro actions assigned with owners and tracked to closure |
